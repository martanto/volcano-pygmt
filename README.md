# volcano-plot
Plotting volcano map using PyGMT (https://www.pygmt.org)

## Requirements
1. Install `uv` python package manager from https://docs.astral.sh/uv/getting-started/installation/
2. Install Generic Mapping Tools (GMT) from https://www.generic-mapping-tools.org/download/. Detailed information regarding GMT https://www.pygmt.org/latest/install.html#error-loading-gmt-shared-library-at
